    Project Summary

    This project enables you to control your LG Television from a computer over a crossed serial cable.
    It is tested in a LG 47LH3000, so if you have issues, please send a patch over with the make and model of your TV and I will integrate!

    Pre-requisites

    • Python (2.6+)
    • pyserial
    • A null modem cable (aka. reversed/crossed serial RS-232) - about £1 from Amazon
